Features

Core Functionality

  • Multi-deck Management - Organize flashcards into themed collections
  • Smart Study Modes - Sequential, shuffled, and difficulty-based study options
  • Category System - Tag and organize cards by custom categories
  • Performance Tracking - Detailed statistics and progress monitoring
  • Offline Support - Full functionality without internet connection

Advanced Features

  • Spaced Repetition - Scientifically optimized review scheduling
  • Difficulty Assessment - Automatic card difficulty adjustment
  • Export/Import - Share decks and backup data
  • Search & Filter - Powerful content discovery tools
  • Dark Mode - Eye-friendly theme options

Installation

Clone Repository

git clone https://github.com/yourusername/xCards.git cd xCards

Install Dependencies

npm install

Start Development Server

npm run dev

Access Application

Open your browser and navigate to http://localhost:5000

Usage Guide

Creating Your First Deck

  1. Click the "Create New Deck" button on the main dashboard
  2. Enter a descriptive name and optional description
  3. Start adding flashcards with questions and answers
  4. Organize cards using categories for better management

Study Sessions

Choose from multiple study modes:

  • Sequential - Study cards in order
  • Shuffled - Random card presentation
  • Not Easy - Focus on challenging cards
  • Not Difficult - Review familiar content

Deployment

GitHub Pages

Deploy the documentation site to GitHub Pages:

node build-docs.js # Upload docs/ folder to your repository # Enable GitHub Pages in repository settings

Production Build

Create optimized production builds:

node build-dist.js # Distribution build node build-docs.js # Documentation build node build-all.js # Complete build process
